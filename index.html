This site was created for a project for Code The Dream. It is utilizing the Star Wars API to person an async fetch with Javascript. What does that mean? Fetch is a way to retrieve information using Javascript from an API. An API is an Application Programming Interface. That's just a fancy way to say that it is a communication between your machine and something that gives you back information. For instance, think of a weather site or app. Your computer or phone connects to the weather API and it gives you back either your local data, or the data of a place that you searched for. Pretty cool!

Fetch in Javascript is a way to send network requests to a server and then load that response info when/where needed. Fetch is a method (which is used as a function) and it requests the web browsers to send a request you make to a URL.
